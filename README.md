# j8-cli

> 支持生成多页应用模板 cli 工具（目前支持生成 html 跟 pug 两种方式），单页应用支持 Vue 模板、React 模板

## Start

```bash
# 全局安装cli
npm i j8-cli -g

# 查看版本
j8 -V

# 创建项目
j8 init
```

## Yarn 安装

```bash
#全局安装
npm i yarn -g

#设置镜像源
yarn config set registry https://registry.npm.taobao.org --global
yarn config set disturl https://npm.taobao.org/dist --global
```
